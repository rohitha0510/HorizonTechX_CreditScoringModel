# Credit Scoring Model

## Objective

Build a machine learning model to predict customer creditworthiness based on financial history and loan-related attributes.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Jupyter Notebook

## Machine Learning Algorithm

* Random Forest Classifier

## Workflow

1. Data Collection
2. Data Preprocessing
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Prediction

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score

## Results

The Random Forest model successfully classified customers based on credit risk and achieved high prediction accuracy.
